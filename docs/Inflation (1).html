Ici on voit qu’en passant la série à l’exponentielle aucune tendance linéaire apparaît donc on sait que l’ajustement par l’exponentielle n’est pas nécessaire. C’est pourquoi nous allons essayer avec un ajustement non-paramétrique.

Ajustement non-paramétrique

#Moyenne mobile de rang 12
p =12
MM4 = stats::filter(x = ind15,  #filtre de moyenne mobile
             filter = c(1/(2*p), # Poids de la première valeur (1 / (2 * p))
                        rep(1/p,p-1), #Poids des valeurs centrées (égaux)
                        1/(2*p)) # Poids de la dernière valeur (1 / (2 * p))
)
MM4=round(MM4,2) #arrondir les moyennes mobiles

x= plot (ind15, xlab = "temps", ylab = "x",col = "grey")

lines(MM4, col = "darkblue",lwd = 2)

Ajustement linéaire

T = length(ind15) #longueur série temporelle

ajust_tend_x.ts = ts(ajust_tend_x$fitted.values,start=c(2015,1), freq=12) # ajustmeent sous forme d'une série temporelle

Saisonnalité

Paramétrique

ind_sans_tend1 = ind15-ajust_tend_x$fitted.values #on enlève la tendance pour se concentrer sur la saisonnalité


# Nombre d'observations
T = length(ind15)
t = 1:T
# Saison de 12 mois
s = 12


sint = sin(2 * pi * t/ s)
cost = cos(2 * pi * t / s)


ajust = lm(ind_sans_tend1~cost+sint) # calcul des paramètres estimés en fonction de sint et cost

ajust.ts = ts(ajust$fit, #(fitted values les estimations pour chaque mois et chaque année)
              start = c(2015,01),
              frequency = 12)
plot(ind_sans_tend1,xlab = "année",ylab="", main = "saisonnalité estimée", type = "o", pch = 20, col = "grey")
lines(ajust.ts,col = "darkblue", lwd = 2)

Non-paramétrique

moy_sais= aggregate(ind_sans_tend1~cycle(ind15), FUN=mean) #Trouver le motif saisonnier

Conclusion :

Pour les années 2022, 2023 et 2024, les prévisions basées sur les données des années précédant la pandémie de Covid-19 (avant janvier 2020) donnent de meilleurs résultats et sont plus représentatives que celles utilisant les données de l’année 2020 marquée par la crise du Covid. Cette observation est confirmée à la fois graphiquement et par les résultats des erreurs quadratiques moyennes.

En effet, l’analyse montre que l’erreur quadratique moyenne du modèle basé sur les données avant Covid est inférieure à celle du modèle comprenant l’impact du Covid. Cela suggère que les prévisions réalisées avec les données d’avant la pandémie sont plus précises.

En d’autres termes on peux dire que la crise du Covid-19 n’a pas eu d’impacts importants sur les indices de prix à la consommation notamment dans les secteurs de l’habillement et des chaussures.
